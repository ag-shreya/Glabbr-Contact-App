## Initialization

1. Load Node modules dependencies

> for frontend

```cmd
    cd frontend
    npm install
```

> Backend Initialization

```cmd
 cd ./backend
```

1. Load Node modules dependencies

   ```cmd
    npm install
   ```

2. To import db

   ```cmd
    npm run start:dev:db
   ```

3. Start the nextjs server
   ```cmd
    npm run start:dev
   ```
    <!-- 
    Initial command (out of the box)
    "start:dev": "nest start --watch",
